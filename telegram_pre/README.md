



相关命令

```angular2html
- 构建并启动容器：docker compose up -d
- 停止并删除容器：docker compose down
- 查看容器日志：docker compose logs -f
- 停止并重新启动容器：docker compose restart
- 列出当前正在运行的容器：docker compose ps
```
